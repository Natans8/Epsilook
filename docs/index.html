Epsilook

Find spells by how they look and sound — a companion for Epsilon WoW

Help

Searching

  • Plain words search names, models, sounds, animations and visual effects at once; a plain number also matches a spell ID.
  • Words match any part of the text, in any order — impact fireball finds cfx_mage_fireball_impact.m2.
  • Quotes make an exact phrase: "fire bolt" matches those words together, in that order. Quotes also keep a model: inside them from becoming a tag.

Field tags

  • Type a prefix like model: (or click a field button) to search one field only. Everything you type then belongs to the tag — spaces included — until you press Enter or Tab.
  • Fields: name: model: sound: anim: fx: mech: id: — sound: and anim: also take an exact SoundKit / AnimKit ID (sound:86835, anim:13839; the old soundkit: and animkit: prefixes still work as aliases).
  • fx: searches visual effects: chains/beams by category word (fx:chain), texture name (fx:shadowlaser) or tint color — a color word (fx:chain red) or a hex code (fx:#00b4ff); dissolves by texture (fx:"dissolve arcane"); glows, ghost effects and model tints by color word or hex (fx:"glow red", fx:"tint green"); desaturation and transparency by percent (fx:"transparency 50%"); freezes and camouflage by word (fx:freeze, fx:camo); full-screen effects by internal name, color word or hex (fx:"screen hex", fx:"screen green"); morphs by NPC name, model name, creature ID or display ID (fx:"morph sheep"); summons by NPC name, creature ID or control type (fx:"summon argi", fx:"summon guardian").
  • model: also matches the usage-category words shown in the Models column — model:missile finds every spell with a projectile model (attached, missile, area, trail, barrage). Category words autocomplete while typing inside an fx:/model: tag, and a tag naming a category exactly floats that category's spells to the top of the results.
  • mech: searches mechanics: spell effect names (mech:resurrect) and aura names on aura-applying spells (mech:mod_stun).
  • Every tag must match (AND), and words inside one tag must match the same file or name — model:6dr + model:crystal means a 6dr model and a crystal model. Exception: several id: tags match any of them (OR) — two id: tags could never both match one spell. Several kit IDs inside one sound:/anim: tag also match any of them.

Excluding

  • A - prefix hides matches instead — -mech:school_damage hides every spell with that mechanic.
  • Four ways: type -model:, click the − half of a field button, click an existing tag's field label to flip it, or type - at the very start of a tag you're editing.

Editing the search bar

  • Enter, Tab or → close the tag you're typing, landing in the gap right after it; ← at the start of a tag closes it into the gap right before it instead.
  • Sitting in a gap (nothing being typed) with an empty input: ←/→ dive into editing the neighboring tag — even before the first one or after the last — so ← then typing inserts text or a new tag right there. Backspace does the same for the tag just before your cursor.
  • Click a tag's text to edit it, its × to remove it. Plain words are edited like normal text.
  • Select across tags and text with the mouse, Ctrl+A, or Shift+←/→ at the text's edge. Ctrl+C copies the selection as query text — model:book note anim:read — ready to paste back into any search bar; typing, Backspace or pasting replaces it.
  • Ctrl+Z undoes any bar edit — tags added, removed or replaced included; Ctrl+Y (or Ctrl+Shift+Z) redoes.

In the results

  • Click any tag to find all spells that use it; Shift-click excludes them instead.
  • ⧉ copies an ID; .lo, /, .mod and the Commands buttons copy ready-to-paste Epsilon commands — hover to preview. Shift-click on those wraps the copy in `backticks`.
  • ▶ on a sound file plays it (streamed from Wowhead); click again to stop.
  • Hovering a texture tag shows the texture; hovering a color swatch shows a large patch of the color with its hex and hue word.

Columns, filters, sharing

  • Hiding a column (toolbar) also excludes it from plain-word search and exports.
  • "Only spells with" narrows the current results (and travels with shared URLs); sorting cycles ▲, ▼, back to relevance. The multi-value columns (Models, Sounds, Animations, Effects, Mechanics) sort by number of entries — the first click puts the most extreme spells on top.
  • The URL always holds your search — click 🔗 Share (next to Export) to copy it, or copy it directly.
  • Add &export=json or &export=csv to any shared URL and opening it downloads the results directly.
