Search syntax

  • Words match any part of the text, in any order — impact fireball finds cfx_mage_fireball_impact.m2.
  • Plain words search names, models and sounds at once.
  • Type a field prefix (name: model: sound: soundkit: animkit: anim: effect: id:) or click a field button to start a tag. Everything you type then belongs to the tag — spaces included — until you press Enter or Tab.
  • Every tag must be satisfied (AND). Words inside one tag must match the same file or name; two tags of the same field mean two separate matches — model: 6dr + model: crystal finds spells using a 6dr model and a crystal model.
  • Click a tag's × to remove it, or its text to edit it. Backspace or ← on an empty bar reopens the previous tag; →/Tab/Enter closes the tag you're typing.
  • Click any tag in the results to search for it. On a tag, ⧉ copies the ID; .lo / / / .mod copy the in-game command (.lookup object, /script PlaySound, .modify animkit).
  • Shift-click any copy button to wrap the copied text in `backticks` (for chat macros).
  • Hiding a column (toolbar) also excludes it from plain-word search and from exports.
