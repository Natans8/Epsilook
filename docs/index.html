Search syntax

  • Words match any part of the text, in any order — 6dr statue finds 6dr_draenei_statue_male01.m2.
  • The tabs choose what your words match. All searches names, models and sounds at once.
  • Prefix a word to target a field regardless of the active tab: name: model: sound: soundkit: animkit: id:
  • Prefixes combine — model:missile name:fire finds fire-named spells using a missile model.
  • Quotes mean an exact file — model:"spells/cfx_mage_fireball_impact.m2" (clicking a tag does this for you).
  • Click any tag in the results to search for it. On a tag, ⧉ copies the ID; .lo / / / .mod copy the in-game command (.lookup object, /script PlaySound, .modify animkit).
  • Hiding a column (toolbar) also excludes it from All-mode search and from exports.
